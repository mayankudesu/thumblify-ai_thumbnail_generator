## 📖 About the Project

**Thumblify** is an AI-powered thumbnail generator website that allows creators to generate high-quality thumbnails instantly using **Google Gemini**.  
The project is built with the **MERN stack and TypeScript**, focusing on performance, scalability, and modern UI/UX.

---

## 🛠 Built With

### Tech Stack

- **MongoDB** – Database
- **Express.js** – Backend framework
- **React.js** – Frontend UI
- **Node.js** – Server runtime
- **TypeScript** – Type safety & maintainability
- **Google Gemini API** – AI thumbnail generation
- **Tailwind CSS** – Responsive styling

---

### Key Features

- 🎨 AI-generated thumbnails using Google Gemini
- ⚡ Fast and responsive user interface
- 🧠 Prompt-based thumbnail generation
- 📱 Fully responsive design
- 🛠 MERN architecture with TypeScript
- 🌐 Deployed on Vercel

## 💻 Getting Started

Follow the steps below to run the project locally.

---

### Prerequisites

- Node.js & npm
- MongoDB (local or cloud)
- Git
- VS Code or any code editor

---

### Setup

Clone the repository:

```bash
git clone https://github.com/mayankudesu/thumblify.git
cd thumblify
```


### Install

Install dependencies:

```bash
npm install
```

### Usage

Start the development server:

```bash
npm run dev
```

Open your browser and go to `http://localhost:5173` (or the port shown in your terminal).


### Deployment

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
